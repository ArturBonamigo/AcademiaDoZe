# 🏋️ Sistema Academia do Zé

🎓 **Projeto Acadêmico — Desenvolvimento de Sistemas 2**

Sistema desenvolvido em C# com o objetivo de simular a gestão de uma academia, permitindo o controle de alunos, planos e atividades.

---

## 🧩 Funcionalidades
- Cadastro de alunos
- Gerenciamento de planos de treino
- Associação de alunos a planos
- Listagem de alunos e seus respectivos planos
- Atualização e remoção de dados

---

## 🚀 Tecnologias
- C#
- Programação Orientada a Objetos (POO)

---

## 🎯 Objetivo
Projeto desenvolvido para prática de conceitos de programação orientada a objetos, organização de código e manipulação de dados em aplicações de console.
